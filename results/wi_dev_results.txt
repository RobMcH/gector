Results on wi_dev_ori

Steps
Generate corrections using:
- predict.py (for full network)
- predict.py with arg --remove_first_layer (for first layer removed)
- pruning_experiment.py (for all but 1 head removed x 12)

Then run errant to convert output to m2 format
errant_parallel -orig 'gector/test_cases/wi_dev_ori.txt' -cor ‘path/to/output.txt’ -out ‘output_file.m2’

Finally use errant lib to compare this generated m2 to the reference m2
python errant/errant/commands/compare_m2.py -hyp output_file.m2 -ref gector/test_cases/wi_dev_ref.m2
